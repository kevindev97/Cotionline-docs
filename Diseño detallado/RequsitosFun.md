# Requisitos Funcionales

## 1. Gestión de usuarios

1.1 Registro de usuarios con estructura uniforme (clientes y proveedores).

1.2 Edición de información personal.

1.3 Cambio y restablecimiento de contraseña.

1.4Inicio y cierre de sesión.

## 2. Administración de negocio

2.1 Registro de información básica de un negocio.

2.2 Subida y validación de documentos obligatorios.

2.3 Gestión de sedes y colaboradores asociados al negocio.

## 3. Proceso de cotización

3.1 Creación, edición, publicación, clonación y eliminación de solicitudes de cotización.

3.2 Compartir solicitudes de cotización.

3.3Comparación y análisis de cotizaciones recibidas.

## 4. Comunicación y soporte

4.1 Envío de notificaciones sobre estado de solicitudes y eventos relevantes.

4.2 Guías y recursos de ayuda para los usuarios.
